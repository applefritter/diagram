Database

A web application program typically interacts with a database.

A database is a software program that may be installed on the server to help with efficiently storing and accessing data.

We can use a database to store Amazon's millions of products as data:

  • description
  • price
  • sizes
  • images

A web application can use this database to dynamically create product pages by populating a pre-made template with product data fetched from the database.
