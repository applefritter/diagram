Serving Content

The simplest way for a server to return a reponse is to return the content of a file on the server. This type of server is known as a Static Server or static asset server.

You can directly send a variety of files over the Internet in this way:

  • text files - *.txt
  • HTML files - *.html
  • images - *.jpg
  • pdfs - *.pdf
  • movies - *.mp4
  • audio - *.mp3
