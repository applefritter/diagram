Server

A web page is sent from a server -- a physical computer much like a PC/laptop.
Servers are connected to the Internet and accept requests for web pages. e.g. requests from web-browsers.

When a request comes in, the server processes that request and sends back some content that will make up the web-page you see.

The simplest way for a server to return a reponse is to return the content of a static file on the server. This type of server is known as a Static Server or static asset server.
