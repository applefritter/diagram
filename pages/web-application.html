Web Application

Serving static files from the filesystem is really fast but it does not scale.

Consider that amazon.com displays millions of products; creating millions of static files would take forever and be hard to maintain.

Instead the web-server can be configured to serve an application instead of static files. An application (web-app) is a software program that responds to web requests (sent from the web-server). An application generates dynamic responses based on the input you feed into it.
