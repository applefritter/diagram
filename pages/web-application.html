Web Application

Serving files from the filesystem is really fast but makes it hard to serve dynamic content.

Consider that amazon.com displays millions of products; creating millions of static files would take forever and be hard to maintain.

Instead the web-server can serve an application (web-app) rather than files. The web-server routes the web-request to the web-application software program.

An application can still interact with the filesystem but it can manipulate the content to generate dynamic responses for per individual request.
